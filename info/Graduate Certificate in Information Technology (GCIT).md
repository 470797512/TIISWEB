## 1. Course Overview

The Graduate Certificate in Information Technology (GCIT) at The Institute of International Studies (TIIS) is a postgraduate qualification designed to provide essential IT knowledge and practical technical skills across core areas of Information Technology.

The program equips students with:

- foundational skills in programming, networks, cybersecurity, and systems development

- practical capability to solve real-world IT challenges

- a pathway into advanced IT studies, including the Master of Cyber Security (MCS)

The GCIT is suitable for:

- students without a computing background

- professionals seeking to transition into IT careers

- individuals aiming to upgrade technical skills or pathway into postgraduate IT programs

## 4. Course Structure

The GCIT consists of 4 core subjects (24 credit points).

The program is:

- a standalone qualification, OR

- a pathway into the Master of Cyber Security (MCS)

Subjects are aligned with the initial stage of the MCS, enabling seamless progression into further postgraduate study .

## 5. Subject Structure

### 5.1 Core Units (4 Units)

| Unit Code | Unit Name                                  |
|-----------|--------------------------------------------|
| MCS501    | Programming Principles                     |
| MCS502    | Introduction to Cyber and Network Security |
| MCS503    | Web and Database Systems                   |
| MCS504    | Software Development Methodologies         |

These subjects provide foundational capability across:

- programming and software development

- networking and cyber security

- web technologies and databases

## 6. Learning Outcomes

Graduates of the GCIT will be able to:

Knowledge

- Apply foundational IT knowledge to evaluate and apply core IT principles

- Understand programming, networking, cybersecurity, and system design concepts

Skills

- Analyse and solve technical problems using appropriate IT tools and methods

- Design and develop basic IT solutions for business environments

- Communicate effectively with technical and non-technical stakeholders

- Work independently and collaboratively to deliver IT solutions

These outcomes support both employment readiness and further academic progression .
